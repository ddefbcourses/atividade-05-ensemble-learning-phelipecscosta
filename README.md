# MNIST — Ensemble Learning

## Como rodar

1. Clone o repositório
   git clone https://github.com/phelipecscosta/mnist-ensemble-learning.git

2. Instale as dependências
   pip install -r requirements.txt

3. Baixe os datasets CSV no Kaggle
   https://www.kaggle.com/datasets/oddrationale/mnist-in-csv
   Coloque mnist_train.csv e mnist_test.csv na mesma pasta do notebook

4. Execute o notebook
   jupyter notebook Phelipe - Ensemble_Learning.ipynb
